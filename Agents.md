## Agent Terminology

- Agent: An autonomous entity that perceives its environment and takes actions to achieve specific goals.
- Task: A specific objective or problem that the agent is designed to solve.
- Environment: The world in which the agent operates, including all entities and their interactions.
- State: A snapshot of the environment at a particular moment in time.
- Action: A move or operation that an agent can perform within the environment.
- Reward: A scalar feedback signal that indicates how well the agent is performing relative to its goals.
- Policy: A strategy that the agent uses to determine which action to take in a given state.
